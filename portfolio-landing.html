Este site

Como este portfólio foi construído

Este portfólio é um arquivo HTML puro — sem frameworks, sem dependências externas. Todas as animações, partículas e interações foram construídas com CSS3 e JavaScript vanilla. O deploy é automatizado via GitHub Actions, publicando direto no GitHub Pages a cada commit na branch main.

🏗️
HTML5 + CSS3
Estrutura semântica e estilização completa com variáveis CSS, glassmorphism, gradientes e animações keyframe — zero dependências.
⚡
JavaScript Vanilla
Canvas 2D para as partículas animadas, IntersectionObserver para scroll reveal, toggle de idioma PT/EN e animação de contadores — tudo sem bibliotecas externas.
🐙
GitHub Actions
Pipeline de CI/CD que detecta push na branch main e faz deploy automático no GitHub Pages. Nenhuma etapa manual de publicação necessária.
🌐
GitHub Pages
Hospedagem gratuita e com CDN global. O site é servido via HTTPS diretamente do repositório, com disponibilidade de 99,9%.
🌍
Bilíngue PT / EN
Troca de idioma em tempo real via atributos data-pt / data-en no DOM. Nenhuma requisição ao servidor — instantâneo.
📱
Responsivo
Layout adaptável a qualquer tamanho de tela com CSS Grid e Flexbox. Funciona perfeitamente em mobile, tablet e desktop.
